Плагин MoneyInst для WordPress

Версия плагина: 2.0.0

Установка:
1. Скопировать содержимое папки "plugins" в папку плагинов WordPress (по умолчанию это "wp-content/plugins/")
2. Перейти в админ панели WordPress на вкладку "плагины", активировать плагин "MoneyInst"
3. Теперь в админ панели WordPress во вкладке "Параметры" появится раздел "MoneyInst" - раздел настроек плагина
4. Чтобы отключаить плагин, необходимо деактивировать плагин в админ панели WordPress на вкладке "Плагины", там же после деактивации можно удалить плагин

При замене ссылок, плагин пытается автоматически определить имя файла. Если ему это не удается и включена опция "Получать имя файла из адреса страницы", то плагин пытается получить имя файла из адреса страницы, указав расширение файла, соответствующее настройке "Тип файла". Если ничего из вышеуперечисленного не дало результат (корректное имя файла не определено), имя файла будет назначено по умолчанию.
